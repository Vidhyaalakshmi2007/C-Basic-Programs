# Basic C Programs

This repository contains a collection of beginner-level C programs that demonstrate fundamental programming concepts such as functions, recursion, arrays, and string processing.

# Programs Included

1. **Binary to Decimal Conversion**
   Converts a binary number to its decimal equivalent.

2. **Simple and Compound Interest Calculation**
   Calculates simple interest and compound interest using functions.

3. **Sum of Digits (Recursion)**
   Finds the sum of digits of a number using recursion.

4. **Prime Number Check**
   Checks whether a given number is prime.

5. **Factorial Using Recursion**
   Computes the factorial of a number recursively.

6. **Maximum and Minimum in an Array**
   Finds the largest and smallest numbers in an array.

7. **Vowel and Consonant Counter**
   Counts the number of vowels and consonants in a string.

8. **Fibonacci Series**
   Prints Fibonacci numbers using recursion.

# Technologies Used

* C Programming Language
* Standard Libraries (`stdio.h`, `math.h`, `string.h`, `ctype.h`)

# How to Run

1. Compile the program using a C compiler.
2. Run the executable file.
3. Enter input as required.

Example compilation:
gcc program_name.c -o program
./program
